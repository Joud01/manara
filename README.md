# 🔦 منارة | Manara

> دليلك قبل ما تدخل الجامعة

كل سنة يوصل آلاف الطلاب الجامعة وهم تايهين بين التخصصات، يختارون بناءً على رأي الأهل أو تخمين الأصحاب أو بحث عشوائي في قوقل. **منارة** موقع بسّطته لأي طالب يحدد تخصصه بناءً على ميوله الحقيقية، يفهم سوق العمل له، ويعرف إجابات أهم الأسئلة قبل لا يسجّل أول مادة.

🔗 **جربي الموقع مباشرة:** [joud01.github.io/manara](https://joud01.github.io/manara/)

---

## ✨ المميزات

### 🧭 البوصلة
اختبار من ٨ أسئلة بسيطة يحدد أقرب مجال دراسي لميول الطالب الحقيقية، مو لتوقعات العائلة أو الأصحاب. النتيجة تربط الطالب مباشرة بتخصصات حقيقية من قاعدة البيانات.

### 🗺️ الخريطة
١٣ تخصص جامعي موزعة على ٦ مجالات رئيسية (تقني، هندسي، طبي، إداري، إبداعي، إنساني)، كل تخصص فيه:
- وصف مختصر لما يُدرَّس فيه
- المسميات الوظيفية المرتبطة فيه
- تقييم الطلب في سوق العمل السعودي
- الرواتب التقريبية
- أفضل الجامعات السعودية لدراسته

### ❓ الأسئلة الشائعة
إجابات مباشرة ومختصرة لأكثر الأسئلة اللي تواجه أي طالب يدخل الجامعة أول مرة، من النسبة الموزونة إلى نظام الساعات المعتمدة.

---

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---|---|
| HTML5 | هيكلة الصفحة |
| CSS3 | التصميم والتنسيق (بدون أي مكتبات خارجية) |
| JavaScript (Vanilla) | منطق الاختبار، الفلاتر، والتفاعل |
| GitHub Pages | استضافة ونشر الموقع |

مشروع خفيف بالكامل، ملف واحد (`index.html`)، بدون أي حزم أو تبعيات خارجية — يفتح ويشتغل مباشرة في أي متصفح.

---

## 📁 هيكل المشروع

```
manara/
└── index.html   ← الموقع كامل (HTML + CSS + JS في ملف واحد)
```

---

## 🚀 التشغيل محليًا

ما يحتاج أي إعداد. فقط:

```bash
git clone https://github.com/joud01/manara.git
cd manara
```

بعدها افتحي `index.html` مباشرة في أي متصفح.

## 👩‍💻 عن المشروع

مشروع شخصي صممته وطورته بالكامل — من الفكرة إلى النشر — **جود الدجاني**، طالبة تقنية معلومات. بدأ كحل بسيط لأختي وهي مقبلة على الجامعة، وتحوّل لمشروع أعمّ يفيد أي طالب يقف بنفس النقطة.
